# Data Structure Animation

Animation for data structure, or the graduation project.

[Live Demo](https://dagnaf.github.io/data-structure-animation) powered by gh-pages. (may be outated, see `branches` | `master)

## Build

```
clone repo.git
cd repo.dir
npm install
npm start
```
